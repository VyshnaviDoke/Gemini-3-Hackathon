Historic Places Explorer is a multimodal AI application that helps users explore and understand historical places through **reasoning-based explanations**. Users can ask questions using text alone or upload images of monuments, ruins, or archaeological sites for deeper analysis.

## üöÄ Features
- Ask questions about historical places using natural language
- Upload images for visual + textual analysis
- Evidence-based historical reasoning (focus on WHY, not just facts)
- Multimodal AI powered by Gemini 3
- Low-latency responses with intelligent model fallback

---

## üß† Gemini 3 Integration
This application uses the **Gemini 3 model family** via the Gemini API:
- **Gemini 3 Pro** for deep multimodal historical reasoning
- **Gemini 3 Flash** as a fallback for faster, reliable responses

Gemini 3 processes text and images together, analyzing architectural elements, materials, symbols, and visible damage to infer historical context and causes.
